  • Four Square

    This is a WIP multiplayer project of a football mini game that we used to play during our school days
    The game features 4 players, each standing on his/her own tile. The goal is to keep the ball moving, with a single touch and make it land on the opponents tile, making it hard for them to hit it.
    I've gone with a custom physics based controller for the character(it started out as an alternative to actually animating the character, but the result was surprisingly good so I've proceeded with it), using configurable joints.
    Applying proper forces to these joints at specified intervals makes the character move around. With the same approach we can make the character perform a kick action.

    Football mini game (Four Square)
